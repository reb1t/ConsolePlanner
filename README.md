Мини-RPG проект в консоли, созданный для закрепления навыков, полученных на курсе Stepik «Основы алгоритмизации и программирования на C#». Покрывает темы: ввод-вывод, переменные, типы данных, операторы, строки, условия (if/else if), циклы (while, for, foreach), break/continue, методы, перегрузка методов, рекурсия, массивы, списки, кортежи, ООП (классы, конструкторы, инкапсуляция, наследование, полиморфизм).
